# 产品定位和特点

## 产品定位

Data Acceleration Kit K-NET（K-Network，网络加速套件）旨在打造一款网络加速套件，提供统一的软件框架，发挥软硬协同优势，释放网卡硬件性能。

<term>K-NET</term>网络加速套件以高性能、易集成为其设计理念，提供面向主流场景的系列化加速库，以开源和闭源相结合的开放方式，培育开发生态，充分满足企业未来业务需求，帮助其数据中心系统转型以更快更好地应对日益激烈的竞争环境，实现与用户的共同成长。

K-NET的网络加速套件目标是面向网络场景，打造一套高性能的协议框架，兼顾性能、以及易用性，并在演进上能满足未来网络场景的需求。

## 产品特点

### 高性能

K-NET网络加速套件提供高性能特性规格：

- 通信协议加速
    - 支持用户态TCP/IP协议栈，将业务TCP/IP网络通信从内核态切换到用户态，提升网络转发效率，进而提升业务性能。
        - Redis单实例单进程场景：单实例场景Redis业务性能相比内核提升75%以上。
        - Redis多实例业务场景：多实例场景Redis业务性能相比内核提升75%以上。
        - 网络benchmark（性能测试）场景：单核带宽相比内核提升300%以上，空载时延降低50%以上。

    - 支持网卡硬件加速功能、支持TSO（TCP segmentation offload，TCP分段卸载）、LRO（Large receive offload，大量接收负载转移）、硬件校验和和Flow Director能力。

### 易集成

K-NET网络加速套件提供对外开放的能力，K-NET通信加速子系统提供标准的对外Socket接口、业务无感接入进行加速、并且结合DPDK社区能力提供报文统计和抓包工具，支持开发者快速应用、迁移。

- 通信协议加速
    - 提供标准POSIX API接口重定向，业务无侵入式修改，业务无感接入。
    - 提供LD\_PRELOAD接入方式，由用户在业务进程启动时使用用户态TCP/IP协议栈。

- 平台能力开放

    融入生态，提供统一的调试工具供用户和开发者快速应用。
