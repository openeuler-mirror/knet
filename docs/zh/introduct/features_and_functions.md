# 特性和功能

## 通信协议加速

### 用户价值

该特性的价值体现如下。

- 用户态TCP/IP协议栈结合网络硬件卸载能力，提升数据报文的转发性能， 且释放CPU资源，节省服务器节点数量与部署空间需求。
- 用户基于标准POSIX业务软件无需修改，无感进行介入，降低用户软件开发成本。
- 提供零拷贝等扩展Socket接口，可进一步获得更高的性能收益。

### 场景举例

Redis数据库。

### 功能说明

K-NET通信协议加速：

- 支持标准POSIX Socket从内核态替换到用户态。
- 支持高性能用户态TCP/IP协议栈（IPv4）。
- 支持TSO卸载。
- 支持LRO卸载。
- 支持Checksum卸载（TCP）。
- 支持抓包。
- 支持协议栈报文统计信息。
- 支持零拷贝接口。

K-NET通信协议加速用户态TCP/IP协议栈加速特性支持业务模型：

- 支持单进程单线程。
- 支持多进程。
- 支持用户态网卡bond。
- 支持本地环回。
- 支持内核流量转发。
- 支持零拷贝接口。
- 支持应用和协议栈共线程部署。

## 运维管理特性

### 用户价值

K-NET通信协议加速用户态TCP/IP协议栈特性提供和标准DPDK生态对接：

- 提供统计信息获取，用于用户进行运维管理。
- 提供抓包工具，用于用户侧问题快速分析定位。

### 场景举例

K-NET通信协议加速用户态TCP/IP协议栈特性：

- 支持在物理机场景K-NET用户态协议栈场景下报文抓包、业务运维。
- 支持在虚拟机场景K-NET用户态协议栈场景下报文抓包、业务运维。

### 运维管理基础功能说明

K-NET通信协议加速用户态TCP/IP协议栈特性：

- 支持一键式运维信息收集，含硬件信息、软件版本信息、日志信息、配置信息、统计信息。
- 支持查询网卡统计信息。
- 支持查询用户态协议栈统计信息。
- 支持一键式和批量安装、卸载、升级软件。
- 支持抓包。
