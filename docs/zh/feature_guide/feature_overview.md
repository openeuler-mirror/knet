# 特性支持

|特性|描述|支持网卡|支持软件|
|--|--|--|--|
|单进程加速功能|提供高性能用户态协议栈处理能力，支持针对单进程业务场景进行网络加速。|SP670|Redis 6.0.20|
|多进程加速功能|提供多个业务进程并发运行的网络处理能力，支持针对多进程业务场景进行网络加速。|SP670|Redis 6.0.20|
|网卡流量分叉功能|提供流量分叉能力开关，通过流表进程流量分发，支持内核态与K-NET应用的同时使能。|SP670|iPerf3 3.16|
|用户态网卡bond功能|提供基于用户态的网卡聚合能力开关，支持将两个物理网口在用户态进行逻辑绑定，实现带宽叠加，可靠性增强。|SP670|iPerf3 3.16|
|内核流量转发功能|提供内核流量转发功能开关，支持通过用户态协议栈对内核态流量进行转发，支持内核态与K-NET应用的同时使能。主要适用于网卡不支持硬件流量分叉场景。|SP670|iPerf3 3.16|
|零拷贝功能|提供零拷贝定制接口，支持用户调用零拷贝接口，避免应用于用户态协议栈之间的内存拷贝，显著降低CPU开销与内存带宽消耗。|SP670|iPerf3 3.16，Tperf 1.0|
|共线程功能|提供共线程定制接口，支持在业务线程直接驱动协议栈，具有节省转发CPU核的优势。|SP670|Tperf 1.0|
|本地流量环回功能|提供本地流量环回能力，支持K-NET进程与本地进程网络通信的功能。|SP670|iPerf3 3.16|

> [!NOTE]说明  
>示例软件仅支持本章节提供的测试功能，其他功能不保证。
